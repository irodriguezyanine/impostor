# Cómo compartir el link de Impostor Chile por WhatsApp

## Link correcto para compartir

Siempre usa el link completo con **https://**:

```
https://elimpostor.cl
```

## Si algunas personas no pueden abrir el link

### 1. Verifica que el link sea correcto
- Debe empezar con `https://` (no http)
- Debe ser `elimpostor.cl` (sin www, sin espacios)

### 2. Pídeles que lo abran así
- **Opción A:** Toca el link y elige "Abrir en navegador" o "Abrir en Chrome/Safari"
- **Opción B:** Copia el link y pégalo directamente en el navegador
- **Opción C:** Envíales el link por otro medio (SMS, email) para probar

### 3. Posibles causas
- **Caché de WhatsApp:** A veces WhatsApp guarda una versión vieja. Que cierren WhatsApp, borren caché (Ajustes > Almacenamiento > Borrar caché) y vuelvan a intentar.
- **Red/WiFi:** Algunas redes corporativas o públicas bloquean ciertos sitios. Que prueben con datos móviles.
- **Versión de WhatsApp:** Que actualicen WhatsApp a la última versión.

### 4. Formato alternativo para enviar
Si el link no se ve clicable, envíalo así:
```
Juega Impostor Chile gratis: https://elimpostor.cl
```

## Mejoras técnicas aplicadas
- Open Graph optimizado para WhatsApp (imagen 1200x630)
- robots.txt permite a WhatsApp/Facebook indexar el sitio
- Dominio unificado: elimpostor.cl
